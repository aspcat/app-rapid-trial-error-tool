# 系统架构文档

## 1. 项目概述

### 1.1 项目名称
App快速试错工具 (app-quick-validation-tool)

### 1.2 项目类型
Web 应用 - React SPA (Single Page Application)

### 1.3 核心功能
基于"关键词驱动 + 两周闭环 + 数据导向"方法论，帮助开发者高效完成 App 开发试错流程的工具平台。

### 1.4 目标用户
- App 开发者
- 产品经理
- 创业团队

## 2. 技术栈

### 2.1 前端框架
- **React** 18.2.0 - 核心 UI 框架
- **React Router DOM** 6.14.0 - 路由管理
- **Vite** 5.4.8 - 构建工具

### 2.2 UI 组件库
- **Ant Design** 5.16.1 - 企业级 UI 组件库
- **@ant-design/icons** 5.2.6 - 图标库

### 2.3 数据处理
- **XLSX** 0.18.5 - Excel 文件处理

### 2.4 样式
- **Less** 4.2.0 - CSS 预处理器

## 3. 系统架构

### 3.1 整体架构
```
┌─────────────────────────────────────────────────────┐
│                    App.jsx                          │
│              (主应用组件 - 布局管理)                  │
├─────────────┬─────────────┬─────────────────────────┤
│   选题库管理  │   竞品分析   │      需求挖掘           │
│  (TopicLib) │ (Competitor)│  (RequirementMining)    │
├─────────────┴─────────────┴─────────────────────────┤
│              LocalStorage                           │
│           (数据持久化层)                             │
└─────────────────────────────────────────────────────┘
```

### 3.2 模块设计

#### 主应用层 (App.jsx)
- 整体布局：Header + Sider + Content
- 路由导航：左侧菜单切换模块
- 新手引导：Tour 组件
- 数据导出：Export 功能入口

#### 功能模块层
| 模块 | 组件 | 功能 |
|------|------|------|
| 选题库管理 | TopicLibrary | 关键词管理、指标标注、优先级排序 |
| 竞品分析 | CompetitorAnalysis | 竞品信息录入、对比分析 |
| 需求挖掘 | RequirementMining | 需求采集、分类整理 |
| 开发进度管控 | - | 待开发 |
| 测试清单生成 | - | 待开发 |
| ASO优化辅助 | - | 待开发 |
| 数据复盘 | - | 待开发 |

#### 数据持久化层
- 使用浏览器 LocalStorage 存储数据
- 数据结构：
  - `topicLibrary` - 选题库数据
  - `customFields` - 自定义字段配置
  - `competitors` - 竞品数据
  - `requirements` - 需求数据
  - `hasVisited` - 新手引导状态

```

│   └── RequirementMining.less
```

## 4. 核心业务流程

### 4.1 选题库管理流程
1. 用户添加选题关键词
2. 标注市场容量、竞争程度、技术难度等指标
3. 系统自动计算优先级
4. 支持自定义字段扩展
5. 数据导出为 Excel/CSV 模板

### 4.2 竞品分析流程
1. 添加竞品信息（直接竞品/间接竞品）
2. 记录核心功能、优势、不足
3. 综合评分
4. 寻找差异化机会

### 4.3 需求挖掘流程
1. 从选题库和竞品分析中提取需求
2. 需求分类整理
3. 优先级排序

## 5. 核心特性

### 5.1 方法论指导
- 关键词驱动：从用户痛点和市场需求出发
- 两周闭环：快速迭代验证
- 数据导向：基于数据分析决策

### 5.2 数据管理
- 本地存储，无需后端
- 支持自定义字段
- 表格排序、筛选、分页

### 5.3 用户体验
- 新手引导（Tour 组件）
- 响应式布局
- 直观的数据可视化（Tag 颜色标识）
